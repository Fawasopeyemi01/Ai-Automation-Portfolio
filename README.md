## 📊 Case Study — Lead Capture Automation

**Client:** Digital marketing agency
**Challenge:** Manually copying leads from forms into CRM — taking hours each week
**Solution:**
- Built a Zapier workflow connecting their form to their CRM
- Added automatic email notification for every new lead
- Created a Google Sheets log for all leads
- Set up a Slack alert for the team for every new submission

**Result:** Fully automated lead capture system — zero manual data entry. Saving the client 5+ hours every week.
